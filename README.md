# Student-Performance-Analyzer

A menu driven command-line application built in python to manage, analyse, and rank student academic performance.  Built from scratch using core DSA concept.

#Features

• Add student- Enter student details with duplicate roll number prevention
• Show all student- Display complete records for all students
• Search by roll number- Linear search to find a student instantly
• Search by name- case-insensitive name based lookup
• Sort by marks- Bubble sort algorithm to rank student by total marks 
• Subject-wise topper- Separate topper for each subject
• Class average- Calculates average performance across all students
• Pass/Fail report- Count of passed and failed students
• Grade distribution- Shows grade assigned to each other
• Save and load data- JSON-based persistent storage
